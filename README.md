# Calci Tool

A simple calculator website built using HTML, CSS, and JavaScript.

## Features
- Age Calculator  
- BMI Calculator  
- EMI Calculator  
- GST Calculator  
- Percentage Calculator  
- Currency Converter  
- Interest Calculators  

## How to Use
1. Open the project folder
2. Click on `index.html`
3. Use any calculator from the menu

## Tech Used
- HTML
- CSS
- JavaScript
